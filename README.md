# ClarinSinPayWall

Para instalar la extensión desde la Chrome Store:

[https://chrome.google.com/webstore/detail/fhgdeocikifjoabgkjfaemfbmakjhjjk/](https://chrome.google.com/webstore/detail/fhgdeocikifjoabgkjfaemfbmakjhjjk/)
